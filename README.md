Requirements:
 - Python3.x
 - OpenCV 2

To install OpenCV run the following commands in a terminal.
 ```bash
 pip3 install opencv
 ```
 
 To run the program run the following commands in a terminal.

 ```bash
 cd src
 python3 main.py
 ```
