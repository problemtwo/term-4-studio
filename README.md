## Requirements:
 - Python3.x
 - OpenCV 2

## Installing OpenCV:

### Ubuntu/Debian

 ```bash
  sudo apt install python-opencv
 ```

### MacOS

 ```bash
  brew install opencv
 ```
 
## Running the program

 ```bash
 cd src
 python3 make_cascade.py
 python3 create_bg_txt.py
 chmod +x create_samples.bash
 ./create_samples.bash
 chmod +x train_cascade.bash
 ./train_cascade.bash
 ```
